Listing Service

A Development Listing Service for PaySwarm.

This listings service is used to register test assets for sale when developing and testing PaySwarm software. It is primarily provided as a way to help developers build new PaySwarm applications without having to generate and host PaySwarm listings. The listings service is designed to be simple to use:

Create

Mint a new URL on the server by POSTing a JSON-LD listing to that URL. Use a prefix, such as a nickname or project as the base part of the URL. 

curl -X POST -d "{}" http://listings.dev.payswarm.com/example/test

Retrieve

Perform a GET on a previously POSTed URL.

curl -X GET http://listings.dev.payswarm.com/example/test

Update

POST a new JSON-LD document to a previously used URL. 

curl -X POST -d "{}" http://listings.dev.payswarm.com/example/test

Delete

Perform a DELETE on a previously POSTed URL. 

curl -X DELETE http://listings.dev.payswarm.com/example/test
